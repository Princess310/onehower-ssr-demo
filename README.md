# Vue-SSR-Demo

Which is based on [vue-hackernews-2.0](https://github.com/vuejs/vue-hackernews-2.0/)

## Usage
```
# install
$ npm install
```

安装后就可以直接开发
```
# dev
npm run dev
```

单元测试
```
# test
$ npm run test
```

发布
```
# build
$ npm run build
```

```
# start server
$ npm run start
```
